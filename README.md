Project Idea

Learnify is a modern, fully responsive landing page for an online education platform. This single-page website showcases a complete edtech experience with professional design, interactive elements, and strategic content sections that guide users from awareness to conversion. The platform highlights diverse course offerings, expert instructors, student testimonials, and essential conversion tools like newsletter signup.

Objectives

Design a pixel-perfect, mobile-first responsive website using pure HTML/CSS

Implement smooth micro-interactions and hover animations for premium UX

Create a cohesive visual design system with consistent spacing and typography

Demonstrate advanced CSS Grid and Flexbox layouts

Build conversion-focused sections (hero CTA, newsletter, auth buttons)
